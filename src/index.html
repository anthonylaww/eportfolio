PROJECTS

Bepis Bot

Instead of paying for a licenced reaction role bot for the Warwick Card Games server - a subcommunity within Warwick Esports, I created Bepis bot. There is a designated message that members will add a reaction to, in order to receive a role on the server. The roles then give them permissions to write and read messages within the subcommunities - i.e. if you reacted with a Hearthstone emote, you can then see Hearthstone related messages.

Forkify

As part of a JavaScript Udemy course by Jonas Schmedtmann, I learnt to create a web application that involved the use of APIs and modern JavaScript (ES6) techniques. In this course, I learnt about different JavaScript versions' compatabilities with browsers. I also learnt about CORS and recognised the need of a CORS proxy for API usage for non-profitable and independant projects. As an extension of the project, I implemented a better shopping cart function and improved the portion calculation algorithm and more.
